# 📱 Awana Sparks Tracker - Android Tablet Installation Guide

## Overview
The Awana Sparks Tracker can be used on Android tablets in three ways:
1. **Progressive Web App (PWA)** - Recommended, works like a native app
2. **Browser Bookmark** - Simple, quick access
3. **File-based Access** - Offline access from device storage

---

## Method 1: Progressive Web App (PWA) - RECOMMENDED ⭐

This method allows the app to work like a native Android app with offline support.

### Requirements:
- Android tablet with Chrome, Edge, or Samsung Internet browser
- Internet connection (for initial setup only)
- Web server or file hosting service

### Steps:

#### Option A: Using a Web Server (Best Experience)
1. **Upload the app to a web server:**
   - Upload all files from the `awana-tracker` folder to your web hosting
   - Note the URL (e.g., `https://yoursite.com/awana-tracker/`)

2. **On your Android tablet:**
   - Open Chrome browser
   - Navigate to your app URL
   - Tap the menu (⋮) in the top-right corner
   - Select "Install app" or "Add to Home screen"
   - Tap "Install"
   - The app icon will appear on your home screen

3. **Launch the app:**
   - Tap the app icon on your home screen
   - The app will open in full-screen mode like a native app
   - Works offline after first load!

#### Option B: Using GitHub Pages (Free Hosting)
1. **Create a GitHub account** (if you don't have one)
2. **Create a new repository** named `awana-tracker`
3. **Upload all files** from the awana-tracker folder
4. **Enable GitHub Pages:**
   - Go to Settings → Pages
   - Select "main" branch
   - Click Save
5. **Access your app** at: `https://yourusername.github.io/awana-tracker/`
6. **Install on tablet** following Option A steps above

## Method 2: Browser Bookmark (Simple Method)

### Using Local Files:
1. **Transfer files to tablet:**
   - Connect tablet to computer via USB
   - Copy the entire `awana-tracker` folder to:
     - `Internal Storage/Documents/awana-tracker/` or
     - `Internal Storage/Download/awana-tracker/`

2. **Open in browser:**
   - Open Chrome on your tablet
   - Type in address bar: `file:///storage/emulated/0/Documents/awana-tracker/index.html`
   - Or navigate using Chrome's file browser

3. **Create bookmark:**
   - Tap the star icon to bookmark
   - Name it "Awana Tracker"
   - Save to bookmarks or home screen

### Using Cloud Storage:
1. **Upload to Google Drive:**
   - Upload the `awana-tracker` folder to Google Drive
   - Right-click `index.html` → Get link → Anyone with link can view

2. **Access on tablet:**
   - Open the link in Chrome
   - Bookmark for easy access

---

## Method 3: Using a File Manager App

1. **Install a file manager** (if not already installed):
   - ES File Explorer
   - Solid Explorer
   - Files by Google

2. **Transfer files:**
   - Copy `awana-tracker` folder to tablet storage

3. **Open the app:**
   - Navigate to the folder in file manager
   - Tap `index.html`
   - Select "Open with Chrome" or your preferred browser

## 🛠️ Troubleshooting

### App won't install as PWA:
- Make sure you're using Chrome, Edge, or Samsung Internet
- App must be served over HTTPS (use GitHub Pages or web hosting)
- Clear browser cache and try again

### Data not saving:
- Check browser storage permissions
- Don't use incognito/private mode
- Ensure sufficient storage space on device

### Excel import/export not working:
- Requires internet connection for first use
- After first use, library is cached for offline use
- Check file permissions in Android settings

### Touch targets too small:
- Use pinch-to-zoom gesture
- App automatically increases touch targets on tablets
- Try landscape orientation

### App running slow:
- Close other browser tabs
- Clear browser cache
- Restart browser
- Restart tablet if needed
